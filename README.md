# AI Bot Project

AI-інфраструктура для бізнесу: Telegram-бот, автоматизація продажів, збір лідів та управління базою даних.

## Можливості:
- 📋 Меню з послугами
- 🧾 AI‑Аудит бізнесу
- 💎 AI‑Офер
- 📈 AI‑Ідеї росту
- 📢 AI‑Реклама
- 🤖 AI‑Автоворонка
- ⚙️ Налаштування чат‑бота
- 💬 Консультація
- 💸 Ціни
- 📝 Промпт‑менеджер
- 📞 Звʼязатися з менеджером

## Особливості:
- Збір ID користувачів для розсилки
- DeepSeek‑двигун для професійних відповідей
- Розсилка повідомлень через /broadcast
